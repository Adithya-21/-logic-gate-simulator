# Logic Gate Simulator

Minimal project scaffold for a Logic Gate Simulator API.

Files added:

- `api/index.py` — minimal Flask API entrypoint
- `requirements.txt` — Python dependencies
- `vercel.json` — Vercel deployment config

Local run:

```bash
pip install -r requirements.txt
python api/index.py
```

Deploy:

Push this repository to a Git provider and deploy with Vercel.
